## 数据库配置
### 创建数据库用户

创建用户例子如下。登录mongo客户端。然后运行如下命令，用来创建用户。
```
use admin
db.createUser(
     {
         user:"root",
         pwd:"root",
         roles:[{role:"root",db:"admin"}]
    }
)
```
